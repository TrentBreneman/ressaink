# Ressa Ink Portfolio

A modern, high-performance portfolio website for **Ressa**, a Fine Line & Blackwork tattoo artist. This project showcases Ressa's artistic vision, professional experience, and an extensive portfolio of work, while providing a seamless booking experience for clients.

## 🚀 Built With

- **Framework:** [Next.js 14](https://nextjs.org/) (App Router)
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Icons:** [React Icons](https://react-icons.github.io/react-icons/)

## ✨ Features

- **Responsive Design:** Optimized for all devices, from mobile phones to high-resolution desktops.
- **Dynamic Portfolio:** A high-quality gallery of Ressa's tattoo work with a custom lightbox for immersive viewing.
- **Client Booking:** Integrated booking section for appointment inquiries.
- **Interactive FAQ:** Clear answers to common client questions about the tattooing process.
- **Modern Animations:** Fluid, performance-focused entrance and scroll animations.
- **Optimized Performance:** Leverages Next.js Image optimization and server-side features for lightning-fast load times.

## 🛠️ Getting Started

### Prerequisites

- Node.js (v18.0.0 or higher)
- npm, yarn, or pnpm

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/ressaink.git
   ```

2. Navigate to the project directory:
   ```bash
   cd ressaink
   ```

3. Install the dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 📁 Project Structure

- `src/app/`: The Next.js App Router directory containing the main layout and page.
- `src/components/`: Reusable React components (Navbar, Hero, Portfolio, etc.).
- `src/components/ui/`: Low-level UI components like the Lightbox.
- `public/`: Static assets including the hero image, profile photo, and tattoo portfolio images.

## 📝 Scripts

- `npm run dev`: Starts the development server with Hot Module Replacement (HMR).
- `npm run build`: Compiles the application for production.
- `npm run start`: Starts the production build.
- `npm run lint`: Analyzes the codebase for potential errors and styling issues.
